# Project Brief

## 项目名称

ToolBox

## 项目定位

一个基于 .NET MAUI Blazor 的本地开发工具箱，集中承载高频开发辅助功能，减少在多个网站/脚本之间切换。

2026-06 起增加 **Blazor Web App（SSR + Interactive WASM）** 作为第二交付形态：Web 版暂时屏蔽无法跨平台共享的能力，MAUI 保持全功能。

## 目标

- 提供离线可用的常见开发小工具（MAUI）
- 提供浏览器可访问的文本/计算/网络工具子集（Web）
- 统一交互体验与入口（单应用内完成）
- 以较低维护成本持续扩展工具页面（Shared RCL 一处编写，双端复用）

## 非目标

- 不追求云端协作或账号体系
- 不在未明确需求时引入复杂后端服务
- Web 版不强行移植桌面专属能力（OCR、AI、C# Runner、目录同步等）

## 成功标准

- 常见工具均可在 1-2 次点击内到达
- 新工具页面能按既有规范快速接入，并评估 Web 是否开放
- MAUI / Web 构建命令稳定，Windows 本地开发链路顺畅
